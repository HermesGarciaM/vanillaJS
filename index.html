Proyectos de práctica en Vanilla JS

Lista de proyectos de práctica en Vanilla JS

La mejor forma de aprender JavaScript es prácticando y escribiendo mucho código. Por lo anterior y para mi desarrollo profesional, he realizado múltiples prácticas y proyectos buscando practicar, en está página encontrarás listados dichos proyectos con su respectivo enlace, cabe mencionar que todos estos son ideas tomadas de LearnVanillaJS y la lista completa la puedes encontrar aquí.

  • Monsters!: Juego inspirado en la película Monsters Inc. y basado en el viejo juego de PC "Buscaminas". El jugador deberá encontrar los 11 monstruos escondidos tras las puertas dando clic en ellas sin encontrar los tóxicos calcetines.
  • Accordion: Script usado para convertir una colección de encabezados en un acordeon para mostrar y ocultar.
  • Pet Finder App: Usar el API de Petfinder para obtener una lista de mascotas disponibles para adoptar y mostrarlas en una página web.
  • Countdown App: Desarrollar una web app que mantenga una cuenta regresiva hasta 0 desde un tiempo específico.
